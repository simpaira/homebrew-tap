# Simpaira Homebrew Tap

Homebrew casks for [Simpaira](https://github.com/simpaira) apps.

## ItsMyPA — private AI meeting notes for macOS

```bash
brew install --cask simpaira/tap/itsmypa
```

Unsigned build? Skip the Gatekeeper dialog entirely:

```bash
brew install --cask --no-quarantine simpaira/tap/itsmypa
```

More: [github.com/simpaira/itsmypa](https://github.com/simpaira/itsmypa)
